Machine Design Cover, February, 1986

The cover story was about mechanical simulation using kinematics analysis and graphical simulation.  The cover picture is from a dynamic simulation of a typewriter print ball mechanism using Unigraphics running on the D-2300.

Computer-Aided Engineering Cover, June, 1986

Another cover story on mechanical simulation and analysis.  This time the cover image was of the results of a 4-bar mechanism kinematics analysis using Unigraphics Mechanism module.
