Interact, Volume 7 - Number 3, August, 1987

This is another one of those issues that appear to be mis-numbered since it is labled Volume 8, Number 3 which apears to have been issued between Volume 7, Number 2 and Volume 7, Number 4.  This issue included a preview of of the upcoming 5th Annual Special Interest Group Meeting coming up in Boston, MA.  The issue included several user interviews from the spring meeting as well as an update of the current regional user groups.  There was also a short section with user hints.
